# Mapas

Al iniciar el juego, el **menú principal** ofrece Jugar, Crear mapa o Salir.

## Guardar y cargar

Los mapas se guardan en `user://maps/` como archivos JSON.

- **Creador de mapas** (arriba a la derecha): **Guardar** y **Cargar**.
- **Jugar**: lista los mapas guardados; selecciona uno y pulsa **Jugar**.

## Crear un mapa

1. Menú principal → **Crear mapa**.
2. Clic en celda: habitación. Clic en borde entre dos habitaciones: puerta compartida.
3. **Salida**, **Jugador** e **IA** para marcar celdas especiales.
4. **Conectar todo** une todas las habitaciones vecinas.
5. **Guardar** (nombre) o **Jugar** para probar sin guardar.

## Controles del editor

- **Volver** (arriba izquierda): regresa al menú principal.
- **Guardar / Cargar** (arriba derecha).
